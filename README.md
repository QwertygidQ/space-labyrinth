# space-labyrinth
A small 2D game made in Go, where you control a spaceship in a labyrinth

## Building
```
cd src/
go get
cd ..
make
```

## Running
```
./labyrinth
```

## Quick build & run
```
sh run.sh
```

## Controls
`W / Up arrow` -- Go forward

`A / Left arrow` -- Turn left

`D / Right arrow` -- Turn right

`Space` -- Self-destruct!
